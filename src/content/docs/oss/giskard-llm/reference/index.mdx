## Core entrypoints

| Name                                       | Purpose                                                                                              |
| ------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| `acompletion(model, messages, …)`          | Async chat completion on the global default client                                                   |
| `aembedding(model, input, …)`              | Async embeddings when the provider exposes them                                                      |
| `aresponse(model, input, …)`               | Stateful Responses / Interactions style calls when implemented                                       |
| `configure(name, provider=None, **kwargs)` | Register config on the default client ([how-to](/oss/giskard-llm/how-to/named-providers-and-client)) |
| `reset()`                                  | Drop cached configs/providers on the default client                                                  |

`LLMClient` mirrors `acompletion` / `aembedding` / `aresponse` as instance methods sharing the same model-string routing.

### Submodules

- **`giskard.llm.chat`**: Tiny helpers (`user`, `assistant`, `system`, `developer`, `message`) constructing typed chat objects when you prefer constructors over plain dict literals.

### Protocols

| Protocol             | Capability checked before dispatch     |
| -------------------- | -------------------------------------- |
| `CompletionProvider` | `complete(...)` → `CompletionResponse` |
| `EmbeddingProvider`  | `embed(...)` → `EmbeddingResponse`     |
| `ResponseProvider`   | `respond(...)` → `ResponseResult`      |

If you request an operation the resolved provider cannot perform, you get **`UnsupportedOperationError`** (see [Errors](/oss/giskard-llm/reference/errors)).

## Exported types (selection)

Chat completions and tools:

- Completions: **`CompletionResponse`**, **`Choice`**, **`Usage`** — [Chat completions](/oss/giskard-llm/reference/completions)
- Messages: **`ChatMessageParam`**
- Tooling: **`ToolDefParam`**, **`FunctionDefParam`**, **`ToolCall`**, **`ToolCallFunction`**, **`FunctionCallOutputParam`**

Embeddings:

- **`EmbeddingResponse`**, **`EmbeddingData`**, **`EmbeddingUsage`** — [Embeddings](/oss/giskard-llm/reference/embeddings)

Responses API:

- **`ResponseResult`**, **`ResponseOutputItem`**, **`ResponseOutputText`**, **`ResponseOutputFunctionCall`** — [Responses / Interactions](/oss/giskard-llm/reference/responses)
